Welcome to my Facebook

This is still a work in progress all functionality is not yet set in place.

Current Functionality:

  • You can create an account and your username will be what you set your first name as and you password will be what you set your password as. You must be over the age of 13 to create a account and must select a gender
  • You can use the account you create to 'login'. At the moment it will only give you a alert that you have logged in. It wont take you to any page yet. There is also a fun little animation when you click on the username input and password input.
  • You can hover over the Recent Logins accounts to see a custom animation I put in place.
